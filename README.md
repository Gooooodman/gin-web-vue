## 快速开始

```
git clone https://github.com/piupuer/gin-web-vue
cd gin-web-vue
# 安装依赖文件, 可以使用cnpm镜像加速
# npm install -g cnpm --registry=https://registry.npm.taobao.org
# cnpm install
npm install
# 本地调试
npm run serve
# 正式发布
npm run build:prod
```

> 启动成功之后, 可在浏览器中输入: [http://127.0.0.1:10001](http://127.0.0.1:10001), 若不能访问请检查node依赖是否安装成功
